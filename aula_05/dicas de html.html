Dicas de HTML

Exemplos simples e úteis. Copie o código e use em suas páginas.

Estrutura básica do documento HTML5

Funcionamento: define o esqueleto da página que o navegador entende.

         <!DOCTYPE html>
         <html lang="pt-BR">
         <head>
         <meta charset="utf-8" />
         <meta name="viewport" content="width=device-width, initial-scale=1" />
         <title>Minha página</title>
         </head>
         <body>
         <h1>Olá, mundo!</h1>
         <p>Bem-vindo ao meu site.</p>
         </body>
         </html>

Imagem com texto alternativo

Funcionamento: o atributo alt descreve a imagem; width ajusta o tamanho.

         <img 
         src="https://via.placeholder.com/640x360" 
         alt="Paisagem com montanhas ao pôr do sol" 
         width="320" />

Lista ordenada (passo a passo)

Funcionamento: use <ol> para a lista e <li> para cada item.

         <ol>
         <li>Preparar os ingredientes</li>
         <li>Pré-aquecer o forno</li>
         <li>Assar por 25 minutos</li>
         </ol>

Tabela simples

Funcionamento: <table> contém a tabela, <tr> é a linha, <th> o cabeçalho e <td> a célula.

         <table>
         <thead>
         <tr>
         <th>Produto</th>
         <th>Preço</th>
         </tr>
         </thead>
         <tbody>
         <tr>
         <td>Biscoito</td>
         <td>R$ 5,00</td>
         </tr>
         <tr>
         <td>Chocolate</td>
         <td>R$ 7,50</td>
         </tr>
         </tbody>
         </table>
